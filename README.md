# The Spinize Project

Open Source Subsonic Client for your audio/music library.


# GOALS

- make using Navidrome and other subsonic servers easy with this Client.
- Offline mode (local downloads)
- theme customisations made easy
- automatically fetch synced lyrics, have synced translations (example japanese to english), and finally pronounciation (example romaji for japanese).
- easily change artworks, authors and all using a web interface (useful if there are missing album covers or misinterpreted authors : with "," for example or "Various Artists" instead of "Artist A, Artist B")
